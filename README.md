# 说明

> ### **NativeScript之美，就是你可以用javascript，xml，和css写出原生ios和安卓app，而不用触及 Swift, Objective-C, 或者 Java**





本教程首先对NativeScript基础部分导入讲解（主要翻译官网新手教程），
然后针对web开发者实例讲解NativeScript的重点知识，
最后我们将从零开始构建几个小项目。
